# SuwaSystem仕様書

## 基本情報
諏訪地域における地方通貨を作り出すシステム
地方通貨を基軸に小コミュニティーに対して経済圏を提供することを目的とする。

## 機能
- アカウント機能
  - [ ] ユーザーは自分のアカウントを作成することができる
  - [ ] アカウントはモバイルタイプとカードタイプが存在する
- コミュニティー機能
  - [ ] ユーザーはコミュニティーを作成することができる
  - [ ] コミュニティーを作成できるユーザーはモバイルタイプのみとする
  - [ ] コミュニティーは通貨を発行し下記項目を定めることができる
    - 基本通貨との交換レート
    - 送金時の手数料
    - 手数料が振り込まれるオーナーウォレット
- 決済機能
  - [ ] ウォレットからウォレットへ送金(決済)を行うことができる
  - [ ] 基本通貨との交換を行うことができる
  - [ ] モバイルタイプの場合はQRコードを読み込むことによって送金金額を指定する
  - [ ] カードタイプの場合は端末(M5Stack Core2)の端末を用いて金額を設定し、送金する
